# Bioseq-analisis
 Bioseq Analysis
 
Proyecto de análisis de secuencias de ADN utilizando Python y Biopython.

  Funcionalidades
- Lectura de archivos FASTA
- Cálculo de GC content
- Transcripción ADN → ARN
- Traducción a proteínas
- Identificación de ORFs

  Estructura
- data/: archivos de entrada (FASTA)
- src/: código principal
- results/: resultados generados

  Instalación
pip install -r requirements.txt

 Uso
python src/main.py

 Ejemplo de salida
ID: secuencia_1  
Longitud: 36  
GC%: 50.0  

 Tecnologías
- Python
- Biopython
